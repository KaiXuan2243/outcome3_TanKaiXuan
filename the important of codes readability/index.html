Explain the importance of codes readability.

Code readability is important for a code, it is the one of the factor to write a good quality code. Code readability can easy to understand, debug, maintain, extend, save time and make the code beautiful. Another is people will feel you are proffesional and like to work with you. How to write the readable code? First is Design, have a well structure and give a good name for classes and method. Remember code must clear. Second is consistency of code formatting, braces, name such as classes, methods and variables, and structure. Next is methods and classes cannot too long. Besides that, try not repeat the code, because when the soma code repeat be used will make it easy to understand, it also less cause to read and people will think this programmer lazy to create reusable code.
